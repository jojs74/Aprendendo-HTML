Pra que que serve <aside> </aside> ?

Ele vai representar uma seção dentro da página, que vai ter um conteúdo diretamente relacionado ao conteúdo principal da página.

Ou onde esse aside está posicionado! ele pode estar posicionado no meio de um conteúdo.

Normalmente ele é disposto, a tipográfia dele é disposta como uma cx lateral, mas isso não é uma obrigatoriedade

O elemento ao lado representa uma seção de uma página que consiste em um conteúdo tangencialmente relacionado ao conteúdo ao redor do elemento ao lado e que pode ser considerado separado desse conteúdo.

Essas seções são frequentemente representadas como barras laterais na tipografia impressa.

O elemento pode ser usado para efeitos tipográficos, como citações ou barras laterais, para publicidade, para grupos de elementos de navegação e para outro conteúdo considerado separado do conteúdo principal da página, comentários referente ao conteúdo principal.

Características:

  • Display: Block
  • Categoria: Fluxo e Seção
  • Contido por:Fluxo
  • Filhos: Todos os elementos de Fluxo(h, p, tags do body)
  • Pode conter tds elementos de fluxo:h, p, nav, links, listas,etc..
  • DOM Inteface: Uses HTMLElement
  • Trabalha c/ os atributod globais
  • Pode contar tb um elemento de navegação da página, diferente do elemento principal!
  • Dentro de aside posso ter um <nav>, um menu por exemplo. Que vai ser diferente do menu principal da página
  • Eu posso usar <aside> quando eu quiser relacionar uma explicação de parte de um texto,Quando precisar colocar uma citação.
  • É considerado de forma separado do conteúdo principal
  • Ele não faz parte de fluxo do conteúdo principal. Serve p/ fazer uma explicação, nota, citação relacionando a esse conteúdo ou partes que precisam de explicação do conteúdo principal.
  • Pode ser uma barra lateral, cx lateral, para propagandas, para grupos de elementos de navegação(nave) como um menu secundário.
