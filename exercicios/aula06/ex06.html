Aula#06 - Tag:   <address>   </address>

A tag address serve p/ armazenar e exibir informações de contato

Características:

  • Container address
  • Display:Block
  • Categoria de Fluxo
  • Filhos: Fluxo
  • É uma tag que não tem atributos próprios, ela só usa os atributos globais
  • DOM interface: Uses HTML Element
  • Não posso aninhar tags address, isto é, colocar um address dentro do outro!
  • Não é uma boa prática utilizar tags ("h" - Tít.,Subtít., "p" - Parágrafo - O "p" em alguns casos específicos.) dentro da tag address, como visto no exemplo abaixo↓
  • São conteúdos bastante limitados que podemos utilizar dentro da tag address
  • Não vamos criar uma estrutura muito complexa dentro dessa tag
  • Preferencialmente vamos utilizar address dentro da tag : <footer>

Exemplo:

<footer> O footer é o rodapé da página, do body
<address> container p/ informações de contato
