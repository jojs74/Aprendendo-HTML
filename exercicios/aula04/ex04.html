Aprendendo a criar links

- Tag <a> ancora </a>

- Tag <a> não possuem BLOCK, portanto se eu não usar a tag <br/> p/ quebra de linha, irá ficar tds os links na msm linha, caso eu escreva um link abaixo do outro.

- CSS: O Parâmetro a{display:block;} dentro das tags:(style) no Cabeçalho - Esse parâmetro serve p/ colocar um BLOCK na Tag <a> </a>, já que essa tag ñ possui o block. Tds tag <a> ficarão com block, agora ñ preciso mais usar a tag <br/> em cada linha de link.

- Um link é tratado por ancora

- Utiliza o parâmetro href

- Uso o parâmetro: target = "_blank" p/ abrir o link em uma nova Aba, ela é usada na msms linha do href

Ex: <a href = "http://www.youtube.com/cfbcursos"   target = "_blank" > CFB Cursos HTML </a >

Link p/ conexão web:

Ex: <a href = "http://www.youtube.com/cfbcursos"   target = " _blank " > CFB Cursos HTML </a>
