O < b > bolt < /b > Descrições:

  • Display: Inline (ñ é block)
  • Categoria: Fluxo e Fraseado
  • Filhos: Fraseados

Usa-se o < b > bolt < /b > para destacar palavras chave em <b>Negrito</b>

Jeito certo de usar: < b > bolt < /b >

< p > Texto texto texto texto < b > certo </b> texto texto texto < b > certo < /b > texto texto texto. < /p >

Jeito errado:

< b > Texto texto texto texto texto texto texto texto texto texto < /b >

< b > Texto texto texto texto texto < /b > texto </b> texto texto texto. </b > / / Não tem atributos próprios.

O < b > bolt < /b > É contido pelo texto, isto é, está inserido no texto, por isso dizemos que ele é contido por um elemento fraseado

Os filhos tb são elemento de texto.

O < b > bolt < /b > não tem atributos próprios, trabalha somente com os atributos Globais
